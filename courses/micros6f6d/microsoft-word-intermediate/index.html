Aims

This course will give a better understanding of the facilities available within Word to speed up use and general productivity.

Objectives

By the end of the course the delegate will be able to carry out the following functions

  • Use advanced table techniques
  • Work with long documents
  • Set up house styles
  • Customise auto correct and auto text
  • View multiple documents
  • Insert pictures and use the drawing tools


Summary

Pre-Requisite

A good working knowledge of the subjects covered on the Basic Course, gained either by user experience, or by attendance on the Word Basic Course.
